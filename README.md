# fistdemo

> 一个 Vue.js 的todolist项目，包含日期localstorage等功能

## Build Setup

``` bash
# 导入模块
npm install

# serve with hot reload at localhost:8080 运行代码
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
